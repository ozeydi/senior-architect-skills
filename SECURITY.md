# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project's patterns or examples, please open an issue at https://github.com/ozeydi/senior-architect-skills/issues.

Do not open public issues for critical vulnerabilities. Instead, email the maintainer directly.

## Best Practices

These skills incorporate OWASP-recommended security practices including:

- Input validation at all boundaries
- Parameterized queries to prevent SQL injection
- Password hashing with bcrypt/PBKDF2
- JWT token validation
- CSRF protection patterns
- Rate limiting implementation
- Audit logging for sensitive operations
- Environment-based secret management
