# Senior Architect Skills

This project contains professional development skills in `skills/` directory.

## Available Skills

- **senior-developer-standard** - Production-ready code quality standards, error handling, testing, security
- **clean-architecture** - System design, DDD, layered architecture, dependency inversion
- **performance-optimization** - Caching strategies, algorithm optimization, database tuning

## Usage

When writing code, reference these skills based on the task:

- General development → senior-developer-standard
- System design → clean-architecture
- Performance work → performance-optimization
